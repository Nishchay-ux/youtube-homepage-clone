# YouTube Homepage Clone

A responsive **YouTube Homepage Clone** built using only **HTML5** and **CSS3**.  
This project recreates the front page layout of YouTube with a focus on clean structure, responsive design, and modern CSS techniques.

## Live Demo

🔗 **View Project Here:**  
https://nishchay-ux.github.io/youtube-homepage-clone/

## Preview

A static clone of YouTube's homepage featuring:

- Fixed top navigation bar  
- Search section  
- Sidebar menu  
- Scrollable category tags  
- Responsive video grid layout  
- Dark theme UI  

## Built With

- **HTML5**
- **CSS3**
- **Flexbox**
- **CSS Grid**
- **Media Queries**

## Features

- Fully responsive layout for desktop, tablet, and mobile
- Fixed header and sidebar
- Horizontally scrollable category section
- Video cards with thumbnails, profile pictures, titles, and stats
- Clean dark mode styling inspired by YouTube
- No JavaScript used

- Flexbox alignment and spacing
- CSS Grid layouts
